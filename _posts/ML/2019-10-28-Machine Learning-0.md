# Machine Learning(ML)

Python을 이용한 ML Pre Processing 강의를 정리한 POST입니다.

## 1. Data 주요 개념

- Data 가 가장 중요하다.
- Data 에 대해서 전처리 과정을 통해 의미 있는 Data를 학습을 해야 ML의 예측 성능이 높아진다.
- Data를 모을수록 좋다. (좋은 Data)
- Data가 편향될 수록 성능이 떨어진다.(Overfiting)
- Data를 그래프화하여 특성을 파악하는 것이 중요하다.
